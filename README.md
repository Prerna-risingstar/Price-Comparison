Price Comparison Web App

A smart full-stack system that compares product prices across multiple sources and helps users instantly find the best deals with clean visual insights.

🚀 Tech Stack
Frontend

HTML
CSS
JavaScript / React

Backend

Node.js / Flask
REST APIs

Data Handling

JSON-based aggregation
External API integration (rate-limited)


✨ Features
🔍 Smart product search & comparison
💰 Multi-source price aggregation
📊 Interactive analytics for price trends
⚡ Fast backend data processing
📱 Fully responsive UI/UX
🔄 Dynamic real-time updates (API-driven)
🧠 System Workflow
User Search → Backend API Fetch → Data Cleaning → Price Comparison Engine → UI Display + Analytics
🖥️ Screenshots


⚙️ Installation & Setup
git clone <your-repo-url>
cd Price-Comparison
npm install
npm start
🚧 Important Note

This project uses external APIs for price aggregation. Due to API rate limits, live deployment is not included. 
The application runs fully in a local environment with proper API configuration.
